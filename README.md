# 齒輪齒數計算教學 (Gear Tooth Counting Tutorial)

互動式教學網站，介紹 5 種以影像處理方式計算齒輪「齒數」的方法。

部署網址：<https://jason385.github.io/gear-tooth-counting-tutorial/>

## 本地開發

```bash
npm install
npm run dev
```

## 部署到 GitHub Pages

第一次部署：執行 `./deploy.sh`（會建立 git repo、推上 GitHub、再用 `gh-pages` 發佈到 `gh-pages` 分支）。

之後只要修改完程式碼，跑：

```bash
npm run deploy
```

即可重新發佈。
